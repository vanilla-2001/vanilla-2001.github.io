什么是Web

Web是World Wide Web的简称，广义上可以理解为万维网，狭义上可理解为网页浏览器的网页。
JavaWeb指使用技术解决Web领域问题的技术综合,可以通俗的理解为使用Java去开发程序所需要掌握的相关技术。

Web前台&后台

前端:网页里显示的样式,比如输入框,按钮以及提示信息等。
后台:看不见,确是Web技术的核心。例如接受用户的信息,判断是否正确,以及返回给网页判断结果。

Java Web的应用范围

  • 开发网站
  • ERP系统、财务系统、运营管理系统
  • 游戏后台开发
  • 手机app的后台开发
