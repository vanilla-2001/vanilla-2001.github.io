柱状图：

排列方式

调整柱形 trace 之间的排列方式 figure.update_layout(barmode=”stack”)

update_layout() 方法中，我们通过设置 barmode（柱形排列模式）参数来设置各个柱形 trace 之间的位置关系。可选的参数有：

· stack 堆叠模式 后加入柱形会堆在先加入柱形上面
· group 分组模式 后加入柱形排在先加入柱形右边，用于分组柱状图，默认值
· overlay 重叠模式 后加入柱形直接放在和先加入柱形相同的位置，遮盖先加入的图形
· raletive 相对模式 x值相同时，将y值为负的柱形排在y值为正的柱形的正下方

宽度、颜色

宽度：**figure.update_traces(width=0.4)**设置宽度为0.4
颜色：
figure.update_traces(selector={‘name’:’老会员’},marker_color=’CadetBlue’)
figure.update_traces(selector={‘name’:’新会员’},marker_color=’MediumAquamarine’)

代码示例：

from plotly.graph_objects import Figure,Bar
#引入layout子模块
from plotly.graph_objects import layout
figure = Figure()
months=['1月','2月','3月','4月','5月','6月','7月','8月','9月','10月','11月','12月']
traffics2019=[2279,4228,4728,3641,3655,3675,3938,4075,3884,3036,2838,2621]
traffics2020=[1807,0,0,0,1342,1898,2745,2996,2843,2557,2311,2002]
trace1=Bar(x=months,y=traffics2019,name='2019年客流人数')
trace2=Bar(x=months,y=traffics2020,name='2020年客流人数')
figure.add_trace(trace1)
figure.add_trace(trace2)
figure.update_layout(barmode="group")
figure.update_traces(width=0.4)
figure.update_traces(selector={'name':'2019年客流人数'},marker_color='CadetBlue')
figure.update_traces(selector={'name':'2020年客流人数'},marker_color='MediumAquamarine')
#定义一个字典
font=dict(size=20,color='CadetBlue',family='宋体')
fig_title=layout.Title(text='2019年及2020年健身俱乐部每月客流人数',font=dict(size=20,color ='CadetBlue'),x=0.5)
figure.update_layout(title=fig_title)
figure.show()

饼图：

from plotly.graph_objects import Figure,Pie
industry = ['互联网', '学生', '教育', '金融', '其他']
number = [10281, 5811, 4871, 3305, 2227]
figure = Figure()
#values数值，labels标签，hole环形图(不加这个参数为饼图)
trace=Pie(values=number,labels=industry,hole=0.8)
figure.add_trace(trace)
figure.show()

散点图+导入数据

import pandas as pd
#path两种方式r'或者\\
path=r'C:\Users\28557\Documents\Tencent Files\2855773204\FileRecv\数据可视化-plotly-散点图\数据可视化-Plotly-散点图\奶茶行业分析.xlsx'
data1=pd.read_excel(path,sheet_name=0,header=0)
brand1=data1['品牌']
shop_number1=data1['门店数']
avrg_price1=data1['客单价']
data2=pd.read_excel(path,sheet_name=1,header=0)
brand2=data2['品牌']
shop_number2=data2['门店数']
avrg_price2=data2['客单价']
from plotly.graph_objects import Figure,Scatter
figure=Figure()
trace1=Scatter(x=shop_number1,y=avrg_price1,name='直营模式',mode='markers',marker_color='Pink',marker_size=6,text = brand1)
trace2=Scatter(x=shop_number2,y=avrg_price2,name='加盟模式',mode='markers',marker_color='SkyBlue',marker_size=6,text = brand2)
figure.add_trace(trace1)
figure.add_trace(trace2)
figure.show()
