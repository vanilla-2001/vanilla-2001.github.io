string头文件

发表于2021-03-24|更新于2021-03-26
|阅读量:

getline

作用:读入一行字符串，遇到回车结束
语法: getline(cin,字符串变量名);
getline(cin,字符串变量名,’字符’);当输入与第三个参数相同的字符时，结束

substr

作用:返回一个从指定位置开始，并具有指定长度的子字符串
语法: 字符串名.substr(开始位置,长度)
字符串名.substr(开始位置)

int main(){   
    string s = "12345";
    string s1 = s.substr(3);
    string s2 = s.substr(2, 3);
    cout << s1 << ' ' << s2;
    return 0;
}

输出：45 345

cin.getline

作用:读入一行字符串，遇到回车结束
语法: cin.getline(字符串首地址,个数);

find

语法：
str1.find(str2);从串str1中查找时str2，返回str2中首个字符在str1中的地址
str1.find(str2,n);从str1的第5个字符开始查找str2
str1.find(c);在str1中查找字符c并返回第一个查找到的地址
str1.find(str2, 2 , 2);从str1中的第二个字符开始查找str2的前两个字符

获取字符串长度

语法：字符串变量名.length()
作用：返回字符串长度
