Sort函数

发表于2021-03-24|更新于2021-03-24
|阅读量:

Sort（）函数是c++一种排序方法之一，学会了这种方法也打消我学习c++以来使用的冒泡排序和选择排序所带来的执行效率不高的问题！因为它使用的排序方法是类似于快排的方法，时间复杂度为n*log2(n)，执行效率较高！

头文件：algorithm
语法：

sort(a,a+n,排序方法)

即sort(开始位置，结束位置，排序方法)
排序方法可不写，默认从小到大排序

sort函数没有第三个参数，实现的是从小到大

#include<iostream>
#include<algorithm>
using namespace std;
int main()
{
    int a[10]={9,6,3,8,5,2,7,4,1,0};
    for(int i=0;i<10;i++)
        cout<<a[i]<<endl;
    sort(a,a+10);
    for(int i=0;i<10;i++)
        cout<<a[i]<<endl;
    return 0;
}

加第三个参数实现从大到小排序

bool compare(int a,int b){
    return a>b;
}
sort(a,a+10,compare);

利用c++标准库说明排序方法

语法：

less<数据类型>()//从小到大排序
greater<数据类型>()//从大到小排序

sort(a,a+10,less<int>());
sort(a,a+10,greater<int>());

对字符排序

#include<iostream>
#include<algorithm>
using namespace std;
int main()
{
    char a[11]="asdfghjklk";
    for(int i=0;i<10;i++)
        cout<<a[i]<<endl;
    sort(a,a+10,greater<char>());
    for(int i=0;i<10;i++)
        cout<<a[i]<<endl;
    return 0;
}
