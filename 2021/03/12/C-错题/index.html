C++错题

2021年03月12日 1,079 字 大概 4 分钟

判断

1、通过命名空间可以区分具有相同名字的函数(对)

2、判断数组声明及初始化语句 char str[]={‘H’,’e’,’l’,’l’,’o’};是否正确。(对)

3、C++的运算符按照其操作数的个数可将运算符分为单目运算符和双目运算符。(错)
**

选择

1、关于C++与C语言关系的描述中，（ ）是错误的。(A)
A.C++语言和 C 语言都是面向对象的
B.C 语言是 C++语言的一个子集
C.C++语言对 C 语言进行了一些改进
D.C 语言与 C++语言是兼容的
C++语言起源于C语言。C++是C语言的超集，即C语言是C++语言的一个子集，同时C++语言与C语言是兼容的。C++对C语言作了很多改进，引进了类和对象的概念。但是C语言是面向过程而不是面向对象的。

2、编写C++程序一般需经过的几个步骤依次是（）。(D)
A.编译、调试、编辑、连接
B.编译、编辑、连接、运行
C.编辑、调试、编译、连接
D.编辑、编译、连接、运行

3、在C++中，cin是（）。(C)
A.预定义的函数
B.预定义的类
C.预定义的对象
D.一个标准的语句
cin是输入流对象，负责从键盘获取数据，然后送到内存。

4、关于面向对象程序设计，面向对象程序设计主要考虑的是提高软件的 ▁▁▁▁▁ 。(B)
A.可靠性
B.可重用性
C.可移植性
D.可修改性
面向对象的程序设计通过类的继承与派生机制以及多态性特性，提高了软件代码的可重用性。

5、关于面向对象方法，下面概念中，不属于面向对象方法的是 ▁▁▁▁▁ 。(B)
A.类
B.过程调用
C.继承
D.对象
面向对象方法是一种运用对象、类、封装、继承、多态和消息等概念来构造、测试、重构软件的方法。面向对象方法从对象出发，发展出对象、类、消息、继承等概念。过程调用是面向过程的程序设计方法的概念。

6、命名空间应用于：(A)
A.避免各个不同函数、变量等的名称冲突
B.以上答案都正确
C.在类外定义类的成员函数
D.提高代码的执行速度

7、目标文件的扩展名是（）。(B)
A..cpp
B..obj
C..h
D..exe
源文件.cpp;目标文件.obj;可执行文件.exe在linux平台上，生成的可执行文件一般扩展名为.out
源文件：这是指由源程序和数据构成的文件。通常由终端或输入设备输入的源程序和数据所形成的文件都属于源文件。它通常是由 ASCII 码或汉字所组成的。
目标文件：这是指把源程序经过相应语言的编译程序编译过，但尚未经过链接程序链接的目标代码所构成的文件。它属于二进制文件。
可执行文件：这是指把编译后所产生的目标代码再经过链接程序链接后所形成的文件。

8、下面的这个循环的循环次数是（ ）。
for(int i=0,j=10; i=j=10; i++,j–)(D)
A.1
B.10
C.语法错误，不能执行
D.无限次
判断式为真

9、执行下列语句后，输出结果是（）。(B)

int a=0;
cout<<(a=4+5,a +5),a+25;

A.24
B.14
C.39
D.9
cout<<(a=4+5,a +5),a+25;输出14，a+25丢失
如果是cout<<((a=4+5,a +5),a+25);，输出39

填空题

1、结构化程序设计具有很多优点，是一种面向______的程序设计方法。(过程)

2、假定x和y为double型, 则表达式x=2, y=x+3/2的值为________。(3.000000)
3/2两边都是整型，结果是整数，因为y是double，所以输出3.000000

3、如果a=1，b=2，c=3，d=4，则表达式a&ltb?a:c&ltd?c:d的值为_________。(1)
条件运算符右结合性，从右至左运算

4、表达式 1+4/5+15<7+4%5+(8,10) 的值为_________。(1)
比较运算符优先级小
