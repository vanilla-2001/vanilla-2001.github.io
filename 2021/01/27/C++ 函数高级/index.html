C++ 函数高级

2021年01月27日 819 字 大概 4 分钟

函数的默认参数

在c++中，函数的形参列表中的形参是可以有默认值的。
语法：返回值类型 函数名 （参数=默认值）

#include<iostream>
using namespace std;
int func(int a, int b = 20, int c = 30) {//b,c有默认值
    return a + b + c;
}
int main() {
    cout << func(10) << endl;//因为b,c有默认值，故只需传递a的值
    system("pause");
    return 0;
}

输出：60

#include<iostream>
using namespace std;
int func(int a, int b = 20, int c = 30) {
    return a + b + c;
}
int main() {
    cout << func(10, 30) << endl;//如果有自己的数据，传入自己的数据；如果没有，使用默认值
    system("pause");
    return 0;
}

输出：70

注意事项：

int func2(int a, int b = 10, int c, int d) {//非法，报错，一旦有默认值，右边的参数都要有默认值，左边有无无所谓
    return a + b + c;
}

int func2(int a, int b = 10, int c = 10, int d = 20) {//合法
    return a + b + c;
}

函数声明和函数实现只能有一处有默认参数

int func2(int a = 10, int b = 10);
int main() {
    cout << func2(10, 10) << endl;
    system("pause");
    return 0;
}
int func2(int a = 10, int b = 10) {//非法，报错
    return a + b;
}

函数的占位参数

C++中函数的形参列表里可以有占位参数，用来做占位，调用函数时必须填补该位置
语法：返回值类型 函数名(数据类型){}
在现阶段函数的占位参数存在意义不大，但是后面的课程中会用到该技术

#include<iostream>
using namespace std;
void func(int a, int) {
    cout << "this is func" << endl;
}
int main() {
    func(10, 10);
    system("pause");
    return 0;
}

占位参数也可以有默认参数

#include<iostream>
using namespace std;
void func(int a, int = 10) {//占位参数也可以有默认参数
    cout << "this is func" << endl;
}
int main() {
    func(10);
    system("pause");
    return 0;
}

函数重载-概述

作用：函数名可以相同，提高复用性
函数重载满足条件：
· 用一个作用域下
· 函数名相同
· 函数参数类型不同或者个数不同或者顺序不用
注意：函数的返回值不可以作为函数重载的条件

未满足第三点：

#include<iostream>
using namespace std;
void func() {
    cout << "func的调用" << endl;
}
void func() {
    cout << "func的调用" << endl;
}
int main() {
    func();//非法报错
    system("pause");
    return 0;
}

三点都满足：

#include<iostream>
using namespace std;
void func() {
    cout << "func的调用" << endl;
}
void func(int a) {
    cout << "func(int a)的调用" << endl;
}
int main() {
    func(10);//根据传入的参数，调用第二个函数
    func();//根据传入的参数，调用第一个函数
    system("pause");
    return 0;
}

函数的返回值不可以作为函数重载的条件：

void func(){
    cout<<"func的调用"<<endl;
}
int func(){
    cout<<"func的调用"<<endl;
}//非法，报错

函数重载-注意事项

· 引用作为重载条件

#include<iostream>
using namespace std;
void func(int &a) {//int &a=10不合法，所以传10的时候，不会调用该函数
    cout << "func(int &a)的调用" << endl;
}
void func(const int &a) {
    cout << "func(const int &a)的调用" << endl;
}
int main() {
    int a = 10;
    func(a);//调用不加const的函数
    func(10);//调用加const的函数
    system("pause");
    return 0;
}

· 函数重载碰到默认函数

#include<iostream>
using namespace std;
void func2(int a,int b = 10) {
    cout << "func2(int a,int b)的调用" << endl;
}
void func2(int a) {
    cout << "func2(int a)的调用" << endl;
}
int main() {
    func2(10);//非法，报错，出现二义性
    system("pause");
    return 0;
}

尽量避免这种情况
